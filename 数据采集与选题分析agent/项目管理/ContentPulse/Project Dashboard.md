# ContentPulse Project Dashboard

## 项目目标
- 做一个面向内容运营的内容监控与选题决策工作台
- 支持按分类管理关键词、对标账号和多平台抓取配置
- 支持内容浏览、选题分析、搜索历史、监控设置四个主视图
- 支持抓取结果持久化，方便后续分析与回看

## 当前完成情况
- 页面原型：已完成可用工作台版本
- 顶层结构：已完成 `内容 / 选题分析 / 搜索历史 / 监控设置`
- 分类管理：已支持新增、切换、重命名、删除
- 平台接入：已接入公众号、小红书
- 持久化：已接入 SQLite
- 搜索历史：已支持按查询记录归档与回看
- 选题快照：已支持跟随历史查询保存和展示

## 当前可用能力
- 内容页支持关键词输入、平台切换、日期轴浏览、一键更新
- 小红书和公众号支持通过服务端代理抓取真实数据
- 每个关键词 × 平台最多保留 20 条结果
- 数量口径已统一到“实际可展示数据”
- 搜索历史页支持：
  - 查看历史记录列表
  - 点击某条记录展开当次抓取内容
  - 查看当次选题快照
  - 恢复到内容页或选题页
  - 重新抓取同一条记录
